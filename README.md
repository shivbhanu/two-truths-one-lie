# Two Truths & One Lie

A party game for team events. Each round, three statements about one person appear on a shared screen — two are true, one is a lie. Players vote on their phones to guess who the statements belong to and which one is the lie.

**Works on a local WiFi network.** The host runs the game from a laptop connected to a TV. Players join from their phone browsers — no app install needed.

---

## Requirements

- [Node.js](https://nodejs.org) v16 or later
- All players on the same WiFi network

---

## Step 1 — Gather statements

### Create a Google Form

**Settings to configure before adding questions:**

| Where | Setting | Value |
|---|---|---|
| Settings → General | Collect email addresses | **Verified** |
| Settings → General | Limit to 1 response | On |
| Settings → Responses | See summary charts and text responses | **Off** |

Verified email collection auto-captures the respondent's Oviva address — no name field needed. Player names are derived automatically: `shiv.singh@oviva.com` → **Shiv**.

**Add exactly these 4 short-answer questions:**

| # | Question |
|---|---|
| 1 | Statement 1 |
| 2 | Statement 2 |
| 3 | Statement 3 |
| 4 | Which statement is the lie? (enter 1, 2, or 3) |

Mark all four as required. Share the form link with everyone — including yourself as the host.

### Export to CSV

1. Open the linked Google Sheet for your form responses
2. **File → Download → Comma-separated values (.csv)**

### Convert to game.json

```bash
node scripts/convert-sheet.js responses.csv --host "Shiv" --password your-password
```

| Flag | Default | Description |
|---|---|---|
| `--host` | *(none)* | Your first name as it appears in game (e.g. `"Shiv"` for `shiv.singh@oviva.com`). Excluded from leaderboard scores but still eligible for Most Mysterious. |
| `--password` | `bi2026` | Admin password used to log in on the night. |

This overwrites `data/game.json`. Verify it looks right:

```bash
cat data/game.json
```

## Step 2 — Run the server

```bash
npm install   # first time only
node server.js
```

The terminal prints the local URL and admin password:

```
Two Truths & One Lie running at http://localhost:3000
Admin password: your-password
```

## Step 3 — Run the night

### Connect the laptop to the TV

Mirror or extend the display. Open a browser on the laptop at `http://localhost:3000` and go full-screen (F11 / Cmd+Ctrl+F).

### Admin login

On the laptop browser, click **Host / Admin Login** and enter your admin password.

### Rules screen

The game opens to a full-screen rules overview — leave this on the TV while people arrive and get seated. When everyone is ready, click **Got it — Let's Play!**

### QR / join screen

A large QR code appears with the local network URL (e.g. `http://192.168.1.42:3000`).

Tell players to **scan the QR code** or type the URL into their phone browser. They select their name from the list and tap **Join Game**.

Once everyone has joined, click **Start Round 1 →**.

### Each round

| Step | Admin action | What players see |
|---|---|---|
| Open voting | **Open Voting** | Three statements appear; players vote on WHO + which is the LIE |
| Monitor | Watch the tally count up | Vote submitted / waiting screen |
| Close voting | **Close Voting** | Locked in |
| Reveal | **Reveal Now 🔍** | Correct name, statements labelled LIE / TRUTH, personal result |
| Next | **Next Round →** | Ready for next person |

After the last round, click **See Scores 🏆** to show the final leaderboard.

### Reset

The **Reset** button (top-right) clears all votes and returns to the rules screen, ready for a new game.

---

## Scoring

| Action | Points |
|---|---|
| Correct WHO (guessed the right person) | +2 |
| Correct LIE (guessed the right statement) | +2 |
| Both correct in one vote | +5 bonus (total 5, not 4) |

### End-of-game awards

- **🕵️ Best Detective** — most points overall (host excluded)
- **🎭 Most Mysterious** — their statements fooled the most people (host eligible)

```
├── public/

├── data/
│   └── game.json             # Generated by convert-sheet.js
└── scripts/
    └── convert-sheet.js      # CSV → game.json converter
```

### game.json reference

```json
{
  "adminPassword": "your-password",
  "hostPlayer": "Shiv",
  "players": ["Alice", "Bob", "Shiv", "..."],
  "slides": [
    {
      "statements": ["...", "...", "..."],
      "correctName": "Alice",
      "lieIndex": 1
    }
  ]
}
```

- `hostPlayer` — optional. When set, that player is excluded from leaderboard scores but remains eligible for Most Mysterious.
- `lieIndex` — zero-based (0 = Statement 1, 1 = Statement 2, 2 = Statement 3).
- One slide per player, in any order.

## Troubleshooting

**Players can't reach the URL**
Make sure the laptop and phones are on the same WiFi. The QR screen shows the correct local IP. `localhost` only works on the host machine itself.

**Someone joined with the wrong name**
They need to close the browser, re-open the URL, and choose the correct name. Votes already submitted are tied to the name chosen at join time.

**Wrong player count on the rules screen**
Re-run the converter after updating your CSV, then restart the server.

**Need to redo a round**
Use **Reset** — this clears all votes across all rounds and returns to the start. There is no per-round reset.
